# Geometry Game

A Python game that generates a rectangle with random coordinates and asks the user to guess a point inside it.

## Features
- Random rectangle generation
- User point guessing
- OOP concepts: inheritance, polymorphism, abstraction, and encapsulation

## How to Run
1. Open terminal
2. Go to the project folder
3. Run:
   python main.py
